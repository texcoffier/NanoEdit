NanoEdit

Attention, NanoEdit est un produit expérimental, ne nous en veuillez pas s'il plante...

Récuperez tout d'abord NanoÉdit dans votre compte. Pour cela, exécutez les commandes suivantes (faites un copié/collé) : 

cd		# Pour revenir sur votre répertoire de connexion
mkdir NE	# Pour faire un répertoire pour NanoÉdit
cd NE	        # On y va!
cp /home/exco/NE/stable/Makefile.proto Makefile
/usr/local/bin/make		# Compilation & Édition des liens & Lancement
Cette manipulation n'est à faire qu'une fois.

Premier sujet de TP

Le premier sujet est le même pour toutes les options, c'est pour que vous apreniez à utiliser NanoEdit.

Pour tous les sujets de TP un pointeur vous permet de récupérer un source C contenant le squelette du TP. Il ne vous restera qu'à copier ce squelette dans votre répertoire NE puis à le modifier.

ATTENTION : à chaque fois que vous copiez un nouveau fichier C dans le répertoire d'objet il faut taper /usr/local/bin/make depend pour mettre à jour le Makefile.

Sujets suivants

Si vous avez terminé un sujet en avance, je vous conseille de commencer le sujet suivant. Les sujets n'ont en effet pas le même niveau de difficulté.

Dans tout ces sujets, on vous donne un squelette d'objet que vous devez compléter. La première chose à faire est de rapatrier le squelette, compiler, créer une instance de l'objet sous NanoÉdit et l'éditer afin de voir à quoi ressemble l'objet et demander l'aide sur les différents champs.

Les sujets de TP suivants différent suivant l'options :

  • Option IMAGE
    • Affichage d'objets simples
    • Affichage surface paramétrique
    • Manipulation de repère courant
    • Bresenham
    • Découpage
    • Octree
    • Plan

Les TP sont à montrer sur écran la semaine suivante.

Squelettes utiles pour les TP d'autres matières

Une fois le squelette rapatrié dans votre répertoire et que NanoÉdit a été compilé et lancé, le menu «Création» contient un sous-menu «Squelettes».
  • Intersection de segments de droite.
  • Triangularisation de polygones.
  • Calcul de convexe de polygones.
  • Affiche courbe de Bezier.
  • Calcul/Affiche intersection Bezier/Segment de droite.
  • Ajoute des points à un tableau de points, on peut bien sûr ``empiler'' des noeuds de ce type.
