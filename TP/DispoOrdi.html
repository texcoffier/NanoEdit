Disposition automatique des ordinateurs

Dans la fonction changement, UNE fois par seconde vous allez déplacer TOUS les ordinateurs de la façon suivante~:
  • Pour tous les ordinateurs A : VECTEUR(A) = (0,0,0)
  • Pour toutes les paires de machines (A,B) :
    d = distance(A,B)
    distance_repos = 10/(1+nb_connexion_tcp)
    deplacement = (d - distance_repos)/10
    if ( deplacement<0 || (deplacement>0 && nb_connexion_tcp>0 )
        VECTEUR(A) += vecteur(A,B)/d * deplacement ;
  • Pour tous les ordinateurs A : ajoutez VECTEUR(A) à leur position.
Les déplacement peuvent très bien s'équilibrer et les machines ne plus se déplacer.
