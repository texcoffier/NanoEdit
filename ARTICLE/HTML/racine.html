C'est la racine de toutes les classes.
Elle contient différents champs qui servent à définir la fenêtre d'édition :

  • Classe : Pour afficher la classe de l'objet

  • Apply/Cancel/Ok : Les classiques

  • Trace : Pour choisir le niveau de débuggage

  • Actif : Pour activer/inactiver l'instance

  • D'autres non visibles : Les paramètres, le widget, ...
