Interface 3D : Manipulations sur un point

Vous pouvez modifier sa valeur de manière textuelle comme avant.

Vous remarquez en cliquant sur le point du graphe que son affichage est différent quand il est actif ou non.

Lorsque l'on utilise un raccourci clavier dans une fenêtre graphique pour la création, celle-ci ce fait à l'endroit indiqué.

Pour déplacer le point intéractivement, il suffit de le faire glisser lorsqu'il est actif.

Le point contient plusieurs valeurs (champ), le premier champ est "(x,y,z)" le deuxième est "x", le troisième "y" et le quatrième "z". On peut déplacer l'ensemble des objets actifs au même endroit en appuyant sur '1' pour changer la valeur du premier champ. '2' celle du deuxième, ...

On peut faire apparaître des informations sur le point en cliquant avec le bouton de droite sur le point.
