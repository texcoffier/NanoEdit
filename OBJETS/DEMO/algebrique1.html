Éditeur algébrique : Graphe

Des expressions algébriques plus complexes sont possibles. En fait, toute celle codable par un graphe orienté sans cycle. Ces graphes permettent de coder les contraintes ``simples''. Le modèle affiché peut être écrit sous la forme : 
x = 1 ;
y = 2 ;
Segment(Point(x,y,8),Point(y,x,7))
NanoÉdit ne peut pas manipuler le graphe : x = f(x)
