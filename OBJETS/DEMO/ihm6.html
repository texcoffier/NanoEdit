Interface Homme/Machine : Mise à jour du graphe

Vous pouvez maintenant aller dans les différentes fenêtres d'édition pour modifier les différentes valeurs. Vous remarquerez que chaque modification faite se répercute sur l'ensemble des objets.

Il est possible de cliquer avec le bouton de droite afin d'avoir une aide sur les différents boutons et zones de l'écran. Vous pouvez aussi le faire dans la fenêtre «NanoÉdit» car c'est une fenêtre d'édition de valeur comme les autres (c'est la fenêtre d'édition de l'objet «modele»).
