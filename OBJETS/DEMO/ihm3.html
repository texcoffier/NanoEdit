Interface Homme/Machine : Création

De la même façon, pour créer un objet, on active ses futurs arguments puis l'on indique l'objet à créer.

Si un raccourci clavier est utilisé, c'est l'objet auquel correspondent le mieux les arguments qui sera créé.

Si le menu «Création» est utilisé, c'est l'objet demandé qui sera créé.
