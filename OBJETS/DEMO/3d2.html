Interface 3D : Manipulations sur un segment

Le segment contient plusieurs valeurs :
  • (1) Le point de départ
  • (2) Le point d'arrivé
  • (3) Le milieu du segment (valeur calculée et stockée dans l'objet mais n'aparaissant pas dans la fenêtre d'édition)
  • (*) Le vecteur directeur (valeur calculée et stockée dans l'objet et qui affichée dans la fenêtre d'édition, cette valeur n'est pas modifiable)
  • (4) Le point ``près'' du point de départ (valeur totalement virtuel n'étant stockée nulle part)
  • (5) Le point près du point d'arrivé.
En utilisant les touches du clavier (1, 2, 3, 4, 5), vous pouvez modifier les différentes valeurs. Avec plusieurs segment actifs la touche (2) positionne leur point d'arrivé au même endroit.

Le segment activé s'affiche avec des ``tag''. On peut directement faire glisser les tags pour modifier les différentes valeurs.

Dans le cas du segment, (3) translate le segment et (4)/(5) déplacent l'un des points tout en gardant la même orientation.
