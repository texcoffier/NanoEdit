Installation de NanoÉdit

Pour l'essayer chez vous il faut :
  • UNIX (ça marche au moins avec Linux/HPUX/IRIX)
  • Une commande "make" supportant VPATH et l'option "-I", il faut donc utiliser le "make" du GNU.
  • Les sources version pour GTK1 Version 1.16 obsolette à ne pas installer.
  • Les sources version 2.4 pour GTK2
Procédure d'installation :
  • Modifier la variable BASE dans Makefile.commun et Makefile.proto
  • Tapez make tags depends pour recalculer les dépendances.
  • Tapez make pour compiler (ou make lib pour générer les librairies)
  • Modifier le fichier configuration si les répertoires et bibliothèques ne sont pas aux mêmes endroits.
