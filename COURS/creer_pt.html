Création d'un point

Vous allez dans le menu «création» et choisissez Figures/0d/Point. Le raccourci clavier est indiqué à droite du menu et les noms grisés indiquent que l'on ne peut pas créer l'objet car les paramètres ne sont pas bons.

Méthode avec saisie des coordonnées 3D. Vous positionnez le curseur dans l'une des «vues».

Une fois le curseur au bon endroit, vous pouvez taper 'p3' pour créer le point. (Ne créer pas le point sur l'un des axes vous aurez du mal à le voir). Pour les points suivants, vous pouvez taper '.' pour répéter le dernier raccourci.

Si le point est créé dans une «vue», il est créé à la position du curseur. Si la bascule (toggle) nommée «entier» est activée le point sera créé avec des coordonnées entières.

Si le point est créé dans une vue orthogonale, la coordonnée manquante est mise à 0 sinon elle est calculée de telle manière que le point soit à la même distance de l'observateur que le point vu.
