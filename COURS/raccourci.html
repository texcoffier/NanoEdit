Les raccourcis clavier

Un raccourci clavier est indiqué par le nom d'une touche. Ou bien par une suite de symboles précédés par des modificateurs comme Shft, Ctrl ou Meta (Alt).
  • vu : Vous tapez simplement 'vu'
  • Shft+se : Vous tapez simplement Shift 'se'
  • ...
Les touches enregistrées et correspondant au début d'un raccourci clavier sont affichées dans la fenêtre principale dans le champ historique, ceci vous permet de savoir ce que vous avez déjà tapé.

Les raccourcis peuvent être tapés dans les «vues 3d» ou dans la fenêtre «graphe»

La touche '.' répète le dernier raccourci utilisé.
