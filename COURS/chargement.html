Chargement d'un modèle

Vous indiquez le nom du nouveau modèle dans le champ prévu à cet effet dans la fenêtre nommée "NanoÉdit" (sans taper le «.ne» ni Return) puis vous choisissez «Fichier/Charge». Pour charger le modèle avec les quatres vues 3D choisissez comme nom de modèle «4vues»

Vous pouvez indiquez le nom du fichier sans indiquer le chemin, NanoÉdit le cherchera partout et vous indiquera ou il l'a trouvé.

Pour charger des fichiers existant, vous pouvez utiliser directement le menu «Charge» qui contient la fusion de tous les noms de fichier modèles (extension ".ne") qui se trouvent dans des répertoires d'objet de NanoÉdit.
