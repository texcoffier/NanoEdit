Fenêtre

Toutes les fenêtres NanoÉdit représentent le contenu d'objets et permettent de les modifier.

Pour qu'une modification soit prise en compte, il faut cliquer sur «Applique» ceci est vrai dans TOUS les cas. Sauf :

  • Le déplacement intéractif de l'observateur dans les fenêtres «vue».
  • Dans la fenêtre principale de NanoÉdit.
  • En cas d'utilisation de «Boutons» déclenchant directement des fonctions.
  • Le changement de taille ou le déplacement des fenêtres «vue».
