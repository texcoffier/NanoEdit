Matériaux

Pour changer des paramètres d'un objet on applique les attributs suivant comme les couleurs :
  • Type d'affichage : Ponctuel/Filaire/Facette.
    Influe aussi sur l'affichage POV.
  • Epaisseur trait : Pour l'affichage POV des lignes.
  • Ambiante : Éclairement par autre chose que les lampes.
  • Diffuse : Lumière diffusée dans toutes les directions par l'objets.
  • Spéculaire/Surface : Lumière diffusée dans une direction préférencielle, ainsi que l'état de surface.
  • Réflexion : quantité de lumière réfléchie (pour le lancé de rayons)
  • Réfraction/Indice : quantité de lumière transmise et indice de réfraction du matériaux.

Un modèle montrant l'influence de l'état de surface sur le rendu est dans : «Charge/Cours/materiaux»
