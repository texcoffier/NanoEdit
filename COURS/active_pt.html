Activation d'un point

Toute opération sur un objet implique que l'on active d'abord l'objet. Les deux méthodes les plus simples sont :
  • Vous cliquez sur le pixel représentant le point dans l'une quelconque des vues graphiques.
  • Vous cliquez sur le «point» dans la fenêtre affichant le graphe.
Un point actif est plus gros et le chemin reliant ce point à l'origine est tracé, ceci permet de ``voir'' sa position dans l'espace. Pour désactiver : on reclique dessus.
