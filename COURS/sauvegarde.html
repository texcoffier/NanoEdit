Sauvegarde

Vous indiquez le nom du modèle dans le champ prévue à cette effet (si ce n'est pas déjà fait), puis vous choisissez dans le menu «Fichier/Sauver». Cela ne vide pas l'historique des Undo.

Si c'est un nom de fichier tous simple, il est sauvegardé à partir du répertoire MODELES du répertoire courant. Si le nom est absolu, il est sauvegardé à l'endroit indiqué.

Le menu «Charge» est mis à jour chaque fois que vous sauvez un modèle.
